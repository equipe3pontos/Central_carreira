# Mapa de rotas do protótipo

O site em [reveal-essay-77974683.figma.site](https://reveal-essay-77974683.figma.site) corresponde ao mockup exportado. A **lista completa de caminhos** não está apenas na documentação antiga: está no código — **`Módulo de Carreira (2)/src/app/routes.tsx`** (`createBrowserRouter`). Qualquer rota nova ou renomeada deve **reflectir-se primeiro** nesse ficheiro e **depois** nesta página (*docs-as-code*).

:::warning Escopo da entrega atual

Nesta fase, a entrega e a documentação operacional cobrem **somente o Módulo Colaborador** (`/app/*`). As rotas **RH Tech People** (`/hr/*`) continuam listadas porque existem no protótipo, mas ficam como **referência de descoberta / roadmap**. Não devem gerar backlog de implementação, critérios de aceite ou testes de produção agora.

:::

No `routes.tsx`, o bloco **`path: "app"`** define **todas** as URLs do módulo Colaborador. Não há sub-rotas escondidas noutro ficheiro — o inventário abaixo fecha com o código.

| Métrica | Quantidade | Notas |
|---------|------------|--------|
| **Rotas canónicas `/app…`** | **8** | Uma por URL distinta: índice `/app` + 7 `path` filhos (`perfil`, `pdi`, `pulse`, `pulse/received`, `pulse/sent`, `projetos`, `squads`). |
| **Aliases legados → Colaborador** | **7** | Redirecionam para o mesmo conteúdo que `/app/perfil` … `/app/squads` — **não** existe alias para o *dashboard* em `/` (o *dashboard* é **só** em **`/app`**). |
| **Superfície URL útil para Colaborador** | **15** | 8 canónicas + 7 aliases (para testes e capturas, cada alias dispara um `replace` para a canónica). |

## Aliases legados (redirecionamento)

Implementados como rotas à raiz que montam `RedirectToApp`: em tempo de execução navegam para **`/app` + o mesmo *pathname*** (`replace`).

| Rota antiga | Destino após redirecionamento |
|-------------|-------------------------------|
| `/perfil` | `/app/perfil` |
| `/pdi` | `/app/pdi` |
| `/pulse` | `/app/pulse` |
| `/pulse/received` | `/app/pulse/received` |
| `/pulse/sent` | `/app/pulse/sent` |
| `/projetos` | `/app/projetos` |
| `/squads` | `/app/squads` |

Estas **7 URLs** não são páginas diferentes do ponto de vista de produto — são **compatibilidade de marcadores** ou links antigos sem o prefixo `/app`.
