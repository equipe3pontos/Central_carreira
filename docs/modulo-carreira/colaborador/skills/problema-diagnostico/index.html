Skills — problema e diagnóstico

Gestor diz "melhore comunicação técnica". Colaborador não sabe: apresentações? RFCs? Influência? Sem matriz, feedback vira opinião. Metas PDI ficam vagas. Taxa de conclusão cai 30%. Progressão L5→L6 leva 18+ meses.

Por que diagnóstico falha sem matriz​

Feedback vago: Gestor diz "melhore comunicação técnica" — colaborador não sabe especificamente o quê.

34 potenciais gaps (28 hard + 6 soft), sem priorização: Qual é crítico para L5? Qual é diferencial para L6?

Hard e soft skills em silos: React vive isolada de Comunicação. Ninguém vê que "React Advanced" + "Comunicação Técnica" = "Design de Soluções de nível L5".

Sem régua: "Sou bom em React?" Comparado a L4? L5? L6? Resposta sem contexto é sempre subjetiva.

Feedback não vira ação: Avaliação anual diz "Competente em liderança". Depois? Nada. Não conecta a PDI, não conecta a trilha, não vira meta.

Custo real de autodiagnóstico falho​

SituaçãoResultado
Metas vagas"Melhorar React" = taxa de conclusão 30% menor que "React Advanced Patterns (Sólido→Referência)".
Feedback sem critério"Competente em liderança" não conecta a ação. Avaliação anual → nada acontece.
Soft skills invisíveisSem SMART observável, "comunicação técnica" é opinião. Não vira meta, não muda comportamento.
Progressão subjetivaL5→L6 leva 18+ meses porque promoção depende de "impressão geral", não diagnóstico claro.

Como grandes tech companies resolvem isto​

Google (career ladders), Amazon (leadership principles), Meta (360 feedback → competências), Stripe (IC rubrics) — todos usam modelo de competências estruturado para diagnóstico, feedback, promoção. Por quê?

  1. Clareza: L5 não é opinião — é "Design de Soluções: Referência + Estratégia Técnica: Sólido + ...".
  2. Comparabilidade: Mesma escala para todos — equidade.
  3. Ação: Gap "React Sólido→Referência" + SMART = meta PDI que vira realidade.
  4. Velocidade: Promoção L5→L6 leva 12-14 meses, não 18+, porque gaps são claros.

Métrica que prova sucesso​

Se matriz de competências conectar hard/soft/gaps com SMART observável:

  • Colaborador identifica 3-5 gaps em menos de 5 minutos
  • 80% dos gaps viram meta PDI em 7 dias (vs hoje: metas vagas)
  • L5→L6 reduz de 18+ meses para 12-14 meses (diagnóstico claro = progresso claro)
  • Feedback actionável — colaborador sente que evoluiu
