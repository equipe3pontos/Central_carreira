## Como ler esta página

Cada hard skill tem Esperado (L5 é expectativa) vs Atual (aonde você está). 🔴 marca gap crítico (alta prioridade para L5→L6). 🟡 marca gap médio.

Cada skill conecta a competência do radar (Design, Estratégia, etc). Gaps viram metas PDI.

**Exemplo:** Micro Frontends (Dev→Sólido) = gap de Arquitetura Frontend → meta PDI: "Dominar Module Federation em 3 meses".
