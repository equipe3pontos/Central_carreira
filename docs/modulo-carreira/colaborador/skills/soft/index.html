Soft Skills (/app/perfil · Skills)

Análise de produto

Finalidade​

Documentar o separador Soft Skills dentro de /app/perfil · Skills: competências comportamentais complementares ao radar e às Hard Skills, avaliadas com indicadores SMART para sustentar PDI e progressão L5 → L6 (liderança técnica e IC).

Metodologia no protótipo​

Cada competência comportamental combina níveis Esperado / Atual (escala verbal do mock) com os cinco critérios Specific, Measurable, Achievable, Relevant, Time-bound, de forma a tornar discussões de desenvolvimento acionáveis e auditáveis.

Rastreabilidade​

DestinoRelação
Matriz / radarEixos do radar (ex.: Comunicação Técnica, Liderança) cruzam com estes blocos
Hard SkillsTécnico + comportamental formam o mesmo separador Skills
/app/pdiMetas SMART derivadas destes indicadores
GapsLacunas agregadas (incl. soft quando aplicável)

Limitações do protótipo​

Textos e números são cenário de demonstração; escalas «Proficiente» / «Competente» precisam de definição normativa no PRD ou framework de carreira para produção.

Conteúdo de referência (protótipo)

Matriz de competências · framework L3–L5 (contexto na subaba)​

Este framework conecta as 7 competências-chave (Radar) às Hard Skills técnicas e Soft Skills comportamentais, identificando Gaps que direcionam o PDI e a progressão de carreira L5 → L6. Cada competência é avaliada de forma estruturada para mapear o desenvolvimento individual.

Separadores horizontais no mock: Competências · Hard Skills · Soft Skills · Gaps — ver também Matriz e Hard Skills.

Soft Skills · metodologia SMART​

Competências comportamentais essenciais para L5+, avaliadas através de indicadores SMART (Specific, Measurable, Achievable, Relevant, Time-bound). Estas skills complementam as competências técnicas e são fundamentais para liderança técnica e progressão de carreira.
