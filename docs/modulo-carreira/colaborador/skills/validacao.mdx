# Skills — validação e métricas

> Matriz reduz tempo até diagnóstico (minutos, não horas). Gaps viram metas SMART. Feedback fica menos subjetivo. Promoção L5→L6 tem critério claro.

---

## Hipóteses de sucesso

1. **Clareza:** Colaborador identifica 3-5 gaps críticos em menos de 5 minutos.
2. **Ação:** 80% dos gaps mapeados viram meta PDI em 7 dias.
3. **Progressão:** Tempo L5→L6 reduz de 18+ meses para 12-14 meses.
4. **Satisfação:** Feedback é menos subjetivo; colaborador sente progresso.

---

## Eventos a rastrear

| Evento | Definição | Sucesso |
|--------|-----------|---------|
| `skill_matrix_viewed` | Utilizador abre matriz de competências | 100 views/mês |
| `gap_identified` | Marca gap como crítico/médio/baixo | 80% com pelo menos 3 gaps identificados |
| `pdi_meta_created` | Cria meta PDI a partir de gap | 80% dos gaps críticos viram meta em 7d |
| `feedback_mapped` | Feedback 360 conectado a gap | 70% do feedback mapeado |
| `progression_assessed` | Promoção utiliza matriz como referência | 100% das promoções L5→L6 |

---

## Critério de aceite

- [x] 7 competências bem definidas com descrição clara.
- [x] Escala (0-5) consistente entre todas.
- [x] Modelo Atual vs Esperado renderiza corretamente.
- [x] Cada competência conecta a hard/soft skills específicas.
- [x] Gaps identificados a partir do modelo (Atual menos que Esperado).
- [x] 13 gaps listados com origem (hard/soft), competência, estado.
- [x] Cada gap tem descrição clara: O quê / Por quê / Ação / Evidência / Prazo.
