Matriz de competências (/app/perfil · Skills)

Análise de produto

Finalidade​

Estabelecer um modelo único de competências para o papel de referência (ex.: L5 Frontend): confronto Atual vs Esperado no radar e desdobramento em hard/soft/gaps nas subáreas do mock. É a camada semântica entre feedback qualitativo, Hard Skills, Soft Skills e gaps.

Rastreabilidade​

DestinoUso

/app/pdiMetas devem poder mapear para lacunas identificadas no modelo

Limitações do protótipo​

Detalhe das Soft Skills (SMART): Soft Skills. Inventário dos 13 gaps: Gaps.

Conteúdo de referência (protótipo)

Propósito do framework (texto na tela)​

Este framework liga as 7 competências-chave (Radar) às Hard Skills técnicas e Soft Skills comportamentais, identificando Gaps que orientam o PDI e a progressão L5 → L6. Cada competência é avaliada de forma estruturada para mapear o desenvolvimento individual.

Sub-separadores ou filtros horizontais​

Ordem no mock: Competências · Hard Skills · Soft Skills · Gaps

Inventários: Hard Skills · Soft Skills · Gaps.

Radar de Competências​

Texto de apoio no ecrã: visão consolidada das 7 competências-chave para o papel exemplificado (L5 Senior Frontend Engineer). O radar confronta nível atual versus esperado para a senioridade alvo e sustenta a análise de gaps.

As sete competências-chave (eixos do radar)​

  1. Design de Soluções
  2. Estratégia Técnica
  3. Arquitetura Frontend
  4. Performance & Otimização
  5. Comunicação Técnica
  6. Trabalho em Equipe
  7. Liderança Técnica

Escala do radar (legenda)​

Na visualização tipo spider chart, os marcos numéricos apresentados incluem 0, 2 e 5 no eixo, contrastando sempre Atual e Esperado (dois perímetros ou séries sobre os mesmos eixos).

LegendaSignificado
AtualNível observado ou avaliado para o colaborador
EsperadoNível de referência para a senioridade / papel (ex.: L5 Senior Frontend Engineer)
Representação visual

Os valores pontuais por competência estão no gráfico do protótipo; esta página fixa rótulos, âmbito das sete dimensões e modelo mental Atual vs Esperado para PRD e PDI.
