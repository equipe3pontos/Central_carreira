Skills — validação e métricas

Matriz reduz tempo até diagnóstico (minutos, não horas). Gaps viram metas SMART. Feedback fica menos subjetivo. Promoção L5→L6 tem critério claro.

Hipóteses de sucesso​

  1. Clareza: Colaborador identifica 3-5 gaps críticos em menos de 5 minutos.
  2. Ação: 80% dos gaps mapeados viram meta PDI em 7 dias.
  3. Progressão: Tempo L5→L6 reduz de 18+ meses para 12-14 meses.
  4. Satisfação: Feedback é menos subjetivo; colaborador sente progresso.

Critério de aceite​

  • 7 competências bem definidas com descrição clara.
  • Escala (0-5) consistente entre todas.
  • Modelo Atual vs Esperado renderiza corretamente.
  • Cada competência conecta a hard/soft skills específicas.
  • Gaps identificados a partir do modelo (Atual menos que Esperado).
  • 13 gaps listados com origem (hard/soft), competência, estado.
  • Cada gap tem descrição clara: O quê / Por quê / Ação / Evidência / Prazo.
